# 🛍️ Customer Behaviour Segmentation

An end-to-end **Customer Behaviour Segmentation** project that combines **Python**, **SQL**, and **Power BI** to analyze customer shopping behavior, uncover purchasing patterns, and build an interactive dashboard for business insights.

---

# 📌 Project Overview

This project analyzes customer shopping data to understand customer demographics, purchasing habits, subscription behavior, and product preferences. The workflow includes data cleaning and preprocessing in **Python**, querying and analysis using **SQL**, and interactive visualization in **Power BI**.

The goal is to help businesses identify customer segments, improve marketing strategies, optimize product offerings, and support data-driven decision-making.

---

# 🛠️ Tech Stack

* 🐍 Python (Pandas, NumPy, Matplotlib)
* 🗄️ MySQL
* 📊 Power BI
* ⚡ Power Query
* 📈 DAX
* 📄 CSV Dataset

---

# 🔄 Project Workflow

1. **Data Collection**

   * Imported customer shopping dataset (CSV).

2. **Data Cleaning & Preprocessing (Python)**

   * Handled missing values
   * Renamed columns
   * Checked data types
   * Removed duplicates
   * Performed exploratory data analysis (EDA)

3. **Data Analysis (SQL)**

   * Customer demographics
   * Revenue analysis
   * Product category performance
   * Repeat customer analysis
   * Subscription analysis
   * Discount effectiveness
   * Seasonal purchase trends
   * Payment method analysis

4. **Dashboard Development (Power BI)**

   * Created KPIs
   * Designed interactive visuals
   * Built slicers and filters
   * Developed DAX measures
   * Published business insights
